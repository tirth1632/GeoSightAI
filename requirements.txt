This project is built using Node.js, so dependencies are managed via package.json rather than this requirements.txt file.

To install all required dependencies, simply run:
npm install

The main requirements are:
- Node.js (v18 or higher recommended)
- React
- Vite
- Express (Backend)
- TailwindCSS
- Recharts
- Map tools (@vis.gl/react-google-maps)

For a complete list of packages, please see package.json.
